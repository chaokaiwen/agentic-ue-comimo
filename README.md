# Agentic UE-CoMIMO for 6G — Supplementary Material

Interactive scenario demos and MATLAB reference code accompanying the manuscript
*"Agentic UE-CoMIMO for 6G: From Virtual Antenna Augmentation to AI-Native Virtualized Terminals"*
(submitted to IEEE Communications Magazine).

**Live site:** https://chaokaiwen.github.io/agentic-ue-comimo/

| Content | Path |
|---|---|
| Overview page | `index.html` |
| Supplementary material (full system models & parameters) | `UE_CoMIMO_2_0_Supplement.pdf` |
| Case 1 interactive demo — creator live streaming with CPE assistance | `case1/` |
| Case 2 interactive demo — wearable blind-spot sensing and warning | `case2/` |
| Case 1 MATLAB reference | `matlab/scene_aware_ue_comimo_uplink.m` |
| Case 2 MATLAB reference | `matlab/run_scenario2_sinr_pd_lookup.m` |

The interactive demos are self-contained HTML files that run entirely in the browser.
The MATLAB scripts require base MATLAB only (R2018b+).
